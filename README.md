# Minefut (prototype)

Prototype en Python d'un jeu inspiré de Madfut — pack opening de cartes.

But: créer une démo locale en Pygame qui permet d'ouvrir des packs de cartes (5 cartes) avec une animation simple.

Prérequis
- Python 3.8+
- Windows PowerShell (instructions ci-dessous)

Installation (PowerShell)

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Lancer la démo

```powershell
python main.py
```
